Is it for you?

Ta książka nie jest przeznaczona dla zupełnych żółtodziobów; musisz orientować się w temacie HTML i CSS.

Będziemy rozwijać tę przykładową stronę od etapu prostego, ale poprawnie sformatowanego kodu HTML, do postaci w pełni responsywnej, stopniowo ulepszanej witryny opartej na najnowszych technikach CSS.

Czego się nauczysz?

Każdy rozdział nauczy Cię podstaw poszczególnych zagadnień, a następnie wykorzystamy tę wiedzę do napisania strony, którą właśnie przeglądasz.

Dowiesz się wszystkiego, czego potrzebujesz do pisania w pełni responsywnych, nowoczesnych stron internetowych przy użyciu najnowszych funkcji HTML i CSS. Jakieś jeszcze wątpliwości?

Spis rozdziałów

  1. Podstawowe wiadomości

    W tym rozdziale utworzysz responsywną stronę internetową. Dowiesz się przy okazji, skąd wzięło się RWD, zapoznasz się z problemami, jakie rozwiązuje, a także zgłębisz podstawowe zasady tworzenia projektów responsywnych w HTML i CSS.

  2. Znaczniki HTML

    Dobry kod HTML jest podstawą każdej udanej strony lub aplikacji. Co ważniejsze, HTML umożliwia technologiom asystującym interpretowanie informacji. W tym rozdziale omówimy wszystko od rozpoczynającego dokument znacznika <!DOCTYPE html> po znacznik zamykający.

  3. Zapytania medialne

    Zapytania medialne dają możliwość obsługi ekranów o różnej szerokości, wysokości, preferencjach kolorystycznych, rozdzielczości (DPI) i innych właściwościach. Przyjrzymy się też zapytaniom medialnym CSS4, dotyczącym położenia kursora i rozmiaru wskaźnika, które odnoszą się do obsługi przy użyciu myszy i ekranów dotykowych.

  4. Układy płynne, flexbox i obrazy responsywne

    Flexbox jest najszerzej obsługiwanym systemem CSS do tworzenia układów stron. Po omówieniu składni i przypadków zastosowania właściwych dla flexboksa przejdziemy od razu do rzeczy. Wykorzystamy flexboksa do rozmieszczenia różnych sekcji strony, którą właśnie czytasz.

  5. Układy oparte na CSS Grid

    CSS Grid jest najnowszym systemem tworzenia układów stron. Nowością w tym wydaniu jest cały rozdział poświęcony zrozumieniu możliwości i korzyści płynących z CSS Grid. Nowonabytą wiedzę wykorzystamy do ulepszenia bieżącej implementacji tej strony.

  6. Selektory CSS, typografia, tryby kolorów i nowe funkcje

    Ten obszerny i zróżnicowany rozdział poświęcony jest zagadnieniom takim jak działanie selektorów CSS, obsługa typografii przy użyciu @font-face i fontów zmiennych, posługiwanie się trybami kolorów HEX, RGBA i HSLA, a także dodatkowym kwestiom związanym z obcinaniem tekstu, układami wielokolumnowymi, funkcją calc i zmiennymi środowiskowymi.

  7. Spektakularny wygląd i CSS

    Kolejny rozdział pełen ciekawostek o CSS. Chcesz się dowiedzieć czegoś o cieniach tekstu i pól? Co z wieloma tłami oraz gradientami liniowymi i promienistymi? Powtarzającymi się gradientami? Filtrami CSS w rodzaju rozmycia lub sepii? Maskami i przycinaniem? Własnością mix-blend-mode? O tym wszystkim - i nie tylko - przeczytasz właśnie tutaj. Ponadto na koniec przyjrzymy się własności CSS scroll-snap, umożliwiającej przewijanie elementów do wskazanego punktu - bez kodu JavaScript!

  8. Grafika SVG niezależna od rozdzielczości ekranu

    Grafika SVG jest nieodłącznym elementem zestawu narzędzi programisty. Znajdziesz tutaj omówienie większości praktycznych przypadków użycia grafiki SVG, wstawiania jej bezpośredno do kodu, umieszczania jej w elementach object, a także używania jej w charakterze tła i standardowego obrazu. Przeczytasz też o tworzeniu arkuszy sprite'ów SVG oraz animowaniu i optymalizacji grafik SVG z myślą o ich użytkowaniu w internecie.

  9. Przejścia, transformacje i animacje

    Rozważnie stosowane przejścia, transformacje i animacje potrafią naprawdę ożywić projekt strony. W tym rozdziale zapoznasz się z wszystkimi trzema efektami. Znajdziesz w nim wiele przykładów i będziesz mógł się pobawić kodem, aby lepiej się zaznajomić z tymi technikami.

  10. Formularze

    W tym rozdziale dowiesz się wszystkiego, czego potrzeba o formularzach, w tym m.in. o strukturze kodu uwzględniającej dostępność, różnych elementach formularzy, walidacji formularzy oraz najnowszym CSS, który zapewnia lepszą kontrolę nad wyglądem formularza niż kiedykolwiek dotąd.

  11. Dodatkowe techniki i dobre rady na pożegnanie

    W ostatnim rozdziale znajdziesz podsumowanie omówionych w książce zagadnień oraz omówienie najlepszego podejścia do tworzenia projektu responsywnej strony internetowej. Przeczytasz także o dobrych praktykach tworzenia kodu HTML i CSS responsywnych stron.
